# Essential Skills

The 5 Claude Code skills every builder should have installed, curated by **Josh McCann** ([monopolymccann](https://monopolymccann.com)) for **The AI-Ecom Playbook**.

## Install all 5 (one command)

```bash
git clone https://github.com/joshmccann/essential-skills && cd essential-skills && ./install.sh
```

Then restart Claude Code. That's it.

## What's inside

| # | Skill | What it does |
|---|-------|--------------|
| 1 | **find-skills** | Claude searches the skills marketplace, finds skills relevant to what you're doing, and installs them for you. |
| 2 | **mcp-builder** | Describe any API and Claude scaffolds a full MCP server, so it can connect to almost any tool with no code. Built by Anthropic. |
| 3 | **webapp-testing** (agent browser) | Gives Claude a real browser via Playwright to open pages, click, type, scrape, and test web apps. |
| 4 | **brand-guidelines** | A design system: give Claude your brand style so it makes on-brand assets without re-prompting every time. |
| 5 | **llm-council** | Turns Claude from a yes-man into a council of 5 advisors who pressure-test your decisions and give an honest verdict. |
